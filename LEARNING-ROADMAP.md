# 网站建设学习路线

**创建日期:** 2026-03-18  
**目标:** 系统掌握网站建设方法论，不再盲目试错

---

## 📚 需要学习的核心领域

### 1. 项目管理方法
**问题:** 今天暴露的最大问题——没有项目管理意识

**需要学习:**
- [ ] 敏捷开发基础（Scrum/Kanban）
- [ ] Git 工作流最佳实践（Git Flow）
- [ ] 版本管理策略
- [ ] 任务分解和优先级排序

**推荐资源:**
- 《敏捷软件开发》
- GitHub Learning Lab
- Atlassian Git Tutorial

---

### 2. 网站架构设计
**问题:** 文件组织混乱，不知道什么该留什么该删

**需要学习:**
- [ ] 网站目录结构最佳实践
- [ ] 静态网站架构
- [ ] 资源管理（CSS/JS/图片组织）
- [ ] SEO 基础结构

**推荐资源:**
- MDN Web Structures
- Google Web Fundamentals

---

### 3. 部署与 DevOps
**问题:** 部署流程不清晰，验证不及时

**需要学习:**
- [ ] CI/CD 基础概念
- [ ] Vercel/Netlify 最佳实践
- [ ] 域名和 DNS 管理
- [ ] CDN 原理和缓存策略
- [ ] 部署验证流程

**推荐资源:**
- Vercel Documentation
- 《DevOps 实践指南》

---

### 4. 网站性能优化
**问题:** 不知道如何评估和优化网站

**需要学习:**
- [ ] Core Web Vitals
- [ ] 图片优化（格式、大小、懒加载）
- [ ] 缓存策略
- [ ] 性能监控工具（Lighthouse）

**推荐资源:**
- Google PageSpeed Insights
- Web.dev

---

### 5. 用户体验（UX）基础
**问题:** 设计决策凭感觉，没有依据

**需要学习:**
- [ ] 用户旅程地图
- [ ] 信息架构基础
- [ ] 响应式设计原则
- [ ] 可访问性（Accessibility）基础

**推荐资源:**
- 《Don't Make Me Think》
- Nielsen Norman Group

---

### 6. 电商网站专项
**问题:** 支付、转化、信任建设不懂

**需要学习:**
- [ ] 电商转化漏斗
- [ ] 支付系统集成（PayPal/Stripe）
- [ ] 信任信号设计
- [ ] 产品页面最佳实践
- [ ] 数字产品交付流程

**推荐资源:**
- Shopify Blog
- Baymard Institute（电商 UX 研究）

---

## 📋 学习计划

### 第 1 周：基础夯实
- [ ] Git 工作流系统学习（2 小时）
- [ ] 网站目录结构优化（1 小时）
- [ ] 建立项目管理制度（已完成）

### 第 2 周：部署与运维
- [ ] Vercel 深度使用（2 小时）
- [ ] DNS/CDN 原理（1 小时）
- [ ] 建立监控和告警（1 小时）

### 第 3 周：性能与 UX
- [ ] Lighthouse 性能优化（2 小时）
- [ ] UX 基础原则（2 小时）
- [ ] A/B 测试基础（1 小时）

### 第 4 周：电商专项
- [ ] 支付流程优化（2 小时）
- [ ] 转化漏斗分析（2 小时）
- [ ] 数字产品交付（1 小时）

---

## 🔧 实践项目

**边学边做，每个阶段应用到纹身店网站：**

1. **项目重构** - 用 Git Flow 重新组织开发流程
2. **性能优化** - 用 Lighthouse 评分达到 90+
3. **UX 改进** - 基于 UX 原则优化产品页面
4. **转化优化** - 设计并实施 A/B 测试
